# Claude3pSetup.exe 使用教程

本教程面向**普通用户**:不需要装 Python、不需要会命令行,只要会双击就行。

---

## 一、下载 exe

1. 打开 Release 页面:
   https://github.com/YangYu19921221/claudeDes/releases/latest
2. 在 **Assets** 里下载 `Claude3pSetup.exe`
3. (Windows SmartScreen 可能提示"未识别的应用"→ 点 **更多信息** → **仍要运行**)

> 也可以从 Actions 页面下最新构建的 artifact(zip 包,解压得到 exe)。

---

## 二、前置条件

- **Windows 10 / 11**
- 已安装 **Claude Desktop**,并已**开启 Developer Mode**
  - 打开 Claude Desktop → 设置 (Settings) → 开发者 (Developer) → 开启 Developer Mode
- 一个第三方网关的 **API Key**(`sk-` 开头)

---

## 三、使用步骤

### 1. 双击 `Claude3pSetup.exe` 启动

界面分四块:网关选择、API Key、模型列表、档案设置。

### 2. 选择网关 URL

下拉框三选一(已写死,不可自定义):

- `https://pikachu.claudecode.love`
- `https://dk.claudecode.love`
- `http://154.12.51.83`

### 3. 填 API Key

在输入框粘贴你的 `sk-xxxxxxxx`。

### 4. 点 **拉取模型列表**

工具会请求网关的 `/v1/models`。常见错误:

| 提示 | 原因 |
|------|------|
| 鉴权失败 (401/403) | API Key 错了或被禁用 |
| 网络超时 | 网关不通,换一个 URL 试试 |
| 接口不存在 (404) | 网关不支持模型列表,联系网关方 |

### 5. 勾选要启用的模型

至少勾一个,推荐勾 `claude-sonnet-4-5` / `claude-opus-4` 之类。

### 6. 填档案名

默认 `Default`。**同名档案会被覆盖**(复用同一个 profile id)。

### 7. (可选) 勾选 **写入后重启 Claude Desktop**

勾上后,点完"写入配置"会自动 kill 并重启 Claude.exe,新配置立即生效。

### 8. 点 **写入配置**

成功提示:`配置写入成功`。

配置实际写到:

```
%APPDATA%\Claude-3p\configLibrary\{profileId}.json
%APPDATA%\Claude-3p\configLibrary\_meta.json   (appliedId 指向新档案)
```

## 四、Claude Desktop 装在非 C 盘?

v1.0.1 起已支持。查找顺序:

1. `_meta.json` 里上次缓存的路径
2. 当前正在运行的 Claude 进程(WMIC)
3. Windows 卸载注册表(`HKCU/HKLM\...\Uninstall\Claude`)
4. 默认目录(`%LOCALAPPDATA%\Claude`、`%PROGRAMFILES%\Claude`、`%PROGRAMFILES(X86)%\Claude`)
5. 都找不到 → 弹出文件选择框,你手动指到 `Claude.exe`

手选后路径会写入 `_meta.json`,下次自动复用。

## 五、常见问题

**Q: 双击 exe 没反应?**
任务管理器看 `Claude3pSetup.exe` 是否已在运行;或在命令行跑 `Claude3pSetup.exe` 看错误输出。

**Q: 写入后 Claude Desktop 没切换网关?**
确认 Claude Desktop 已启用 Developer Mode;重启 Claude Desktop;检查 `%APPDATA%\Claude-3p\configLibrary\_meta.json` 的 `appliedId` 是否指向新档案。

**Q: 如何切回官方?**
在 Claude Desktop 的 Developer 设置里关闭 Developer Mode,或把 `_meta.json` 的 `appliedId` 设为 `null`。

**Q: 配置坏了想恢复?**
工具每次写入前会自动备份到 `%APPDATA%\Claude-3p\configLibrary.bak-YYYYMMDD-HHMMSS\`,把它重命名回 `configLibrary` 即可。

**Q: SmartScreen / 杀毒软件报毒?**
PyInstaller 打的 exe 常被误报。可在 Release 页面看到对应 commit hash,自行用 `build.bat` 在本地重新打包验证。

## 六、卸载

直接删掉:

- `Claude3pSetup.exe`(自己下载的位置)
- `%APPDATA%\Claude-3p\` 整个目录(清掉所有档案)
